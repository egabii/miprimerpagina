# Primeros pasos en git

## Que necesitamos?

- Una cuenta en [github](https://github.com/) (podria ser tambien en otra plataforma de desarrollo colaborativo: [gitlab](https://about.gitlab.com/), [bitbucket](https://bitbucket.org/))

- instalar git en nuestra computadora: desde el sitio oficial [git-scm](https://git-scm.com/downloads)

## Creando nuestro primer repo en github

Una vez creada nuestra cuenta en github y que hemos instalado git en nuestra PC. 

Necesitamos crear un repositorio (aka repo) "remoto" en github. 

Los pasos se pueden encontrar en github al crear un nuevo repositorio en tu cuenta

![nuevo repo](./img/new-repo.png)

Al hacer click en el boton verde **NEW** nos manda a la vista para darle un nombre a nuestro repo

![nombrar repo](./img/create-repo.png) 


"Scrolleamos" hasta bajo y vemos otro boton verde para crear nuestro repo

Le damos click y..Listo! creamos un repo en github

![listo repo](./img/done-repo.png)
